# methodology · A metodologia

A forma de trabalhar com IA, em três seções. Narrativa (*porquê/o quê*); o passo-a-passo
prático com comandos está em [`../docs/USO.md`](../docs/USO.md).

- [`01-onboarding/`](01-onboarding/) — **Onboarding de IA:** camadas de contexto,
  `AGENTS.md` canônico, `/setup` e especialização sob demanda.
- [`02-execution/`](02-execution/) — **Execução com SDD:** as 5 fases, gates, Dev Loop e
  subagents.
- [`03-standards/`](03-standards/) — **Padrões e convenções:** versionamento (D1) e
  qualidade no uso de IA (D2), consolidados.

> Cada seção **referencia** os artefatos vivos (`~/.claude/CLAUDE.md`, `.claude/rules/`) em
> vez de duplicá-los — o documento explica, o artefato aplica.
